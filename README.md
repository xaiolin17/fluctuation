# Fluctuation - 市场反转点预测系统

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![PyTorch](https://img.shields.io/badge/PyTorch-1.9%2B-red)
![License](https://img.shields.io/badge/License-MIT-green)

基于深度学习和多时间框架技术指标的高级市场趋势预测系统，支持LSTM和Transformer模型。

## 📋 项目概述

这是一个完整的量化交易预测系统，通过分析纳斯达克100指数(^NDX)的技术指标和波动率特征，使用深度学习模型预测市场趋势反转点。系统包含数据获取、特征工程、模型训练、回测验证等完整流程。

## ✨ 核心特性

### 🚀 多模型架构
- **LSTM模型**: 带注意力机制的长短期记忆网络
- **Transformer模型**: 基于自注意力机制的序列模型  
- **集成学习**: 支持多模型投票集成
- **增量学习**: 支持在线模型更新

### 📊 智能特征工程
- **20+技术指标**: 移动平均、RSI、布林带等
- **多时间框架**: 波动率特征分析
- **自适应阈值**: 智能标签系统
- **向量化计算**: 高性能优化

### 🔧 现实回测系统
- **交易成本模拟**: 0.1%手续费
- **滑点影响**: 0.05%滑点模拟
- **动态仓位管理**: 凯利公式优化
- **风险评估**: 完整绩效指标

## 🛠 安装要求

### 系统要求
- **Python**: 3.8+
- **PyTorch**: 1.9+
- **网络连接**: 金融数据获取
