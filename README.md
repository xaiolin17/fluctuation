# fluctuation

市场反转点预测系统
基于深度学习和多时间框架技术指标的高级市场趋势预测系统，支持LSTM和Transformer模型。

📋 项目概述
这是一个完整的量化交易预测系统，通过分析纳斯达克100指数(^NDX)的技术指标和波动率特征，使用深度学习模型预测市场趋势反转点。系统包含数据获取、特征工程、模型训练、回测验证等完整流程。

✨ 核心特性
🚀 多模型架构
LSTM模型: 带注意力机制的长短期记忆网络

Transformer模型: 基于自注意力机制的序列模型

集成学习: 支持多模型投票集成

增量学习: 支持在线模型更新

📊 智能特征工程
20+技术指标（移动平均、RSI、布林带等）

多时间框架波动率特征

自适应阈值标签系统

向量化计算优化

🔧 现实回测系统

凯利公式仓位管理

完整绩效指标
